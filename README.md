# PowerX Python Engineering Task

For details of the task, see https://powerxai.notion.site/Software-Engineer-Python-805bcc8b246448a1b549b386b95548ab

## Project Setup (Poetry)

This project uses [Poetry](https://python-poetry.org/) for dependency management and packaging. It requires Python 3.11 or higher.
It uses [Pyright](https://github.com/microsoft/pyright) for static type checking. Please ensure that the type checks pass.

Any other setup instructions are deliberately left blank.

## Tech Stack
- Python 3.11+
- [Flask](https://flask.palletsprojects.com/) (web framework)
- Poetry (dependency management)
- (dev) [Pyright](https://github.com/microsoft/pyright). (static type checking)

## Notes
- All dependencies are managed in `pyproject.toml`.
- The main application code is in the `starter_interview_swe_python` package.
